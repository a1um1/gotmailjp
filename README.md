# GOTMAIL.JP Archival

GotMail.jp Flash escape game archival project powered by Ruffle

### Local Deployment

To test the production build locally:

```bash
bun run build
bun run preview
```

The preview will be available at `http://localhost:4173/gotmailjp/`
